# Lo Studio Facile

Sito educativo per studenti delle superiori con materiali organizzati per anni, integrato con Google Classroom.

## Tecnologie
- **Frontend**: React + Next.js
- **Backend**: Node.js + Express
- **Database**: MongoDB
- **Autenticazione**: Google OAuth (Classroom)
